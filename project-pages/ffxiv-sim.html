FFXIV Simulator
Project Details / Key Highlights
This is a simulator for the MMORPG game FFXIV. I created this in hopes of being able to make my own way to practice certain mechanics in boss fights for the game.
  • Built with Unity in C#
  • Includes a raid editor which allows you to re-create or make your own mechanics.
  • Created with saving/loading in mind using JSON.
