SFML - 2D Platformer
Project Details / Key Highlights
A short Mario-like platformer game. I made this for the Harvard CS50 Course's Final Project.
  • Built mostly from scratch using C++ and the SFML library
  • Able to read a text file to make an entire level.
  • Code is made for easy future additions/features.
