Smash Guys
Project Details / Key Highlights
My final project in school done completely solo. I decided to go with a smash-like game on mobile/pc and including networking so I could take the time to learn many different aspects of creating an online game.
  • Built with Unity in C#
  • Available on PC / Android
  • Supports local multiplayer using UNET
  • Developed a modular combat system that supports unlimited character and combo growth
  • Successfully imitated many of the general systems found in the original game.
