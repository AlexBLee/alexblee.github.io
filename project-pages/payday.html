PAYDAY: Crime War
Project Details / Key Highlights
This was my first commercial game I worked on that started from pre-production. I was essentially an all-rounder working in most aspects of the game. Some things I've done are:
  • Converting the Unity animation system to use Animancer, an animation package.
  • Adding network functionality to certain gameplay features.
  • Profiled and optimized the game to increase framerates.
  • Added controller support using Rewired.
  • Implemented the UI functionality for getting the character skill trees working.
  • Using Playfab for players to store their save data.
  • Symbolicated, analyzed and diagnosed game crashes for the team to look at.
