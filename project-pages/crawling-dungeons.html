Crawling Dungeons
Project Details / Key Highlights
Created in 2020, this was my biggest side project I've done. Inspired from turn-based games like Final Fantasy, I created a very minimalistic experience for mobile players that I could think of.
  • Built with Unity in C#
  • Released on itch.io for the browser, and the Play Store for Android.
  • Created many modular systems for good code encapsulation.
  • Includes tools in the Editor to make your own enemies and levels.
