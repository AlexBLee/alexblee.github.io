Tankapalooza
Project Details / Key Highlights
An endless wave shooter involving toon tanks. This was my first project in Unreal Engine.
  • Built with C++ and Blueprints.
  • Features an buff system that allows for easy and simple implementation of new buffs.
  • Uses Unreal's AI system to create simple AI.
