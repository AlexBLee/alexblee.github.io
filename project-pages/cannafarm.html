Cannafarm
Project Details / Key Highlights
Cannafarm is the first game I've worked on in my professional career. It was simply a Farmville-like game that involves growing weed in your farm. Working on this game I have:
  • Refactored the loading screen back-end code to be more organized and a bit more efficient.
  • Converted the current buying and selling market system to use Cloud Firestore.
  • Used Playfab and asynchronous programming to improve the functionality for players buying/selling items.
